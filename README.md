# C-Sharp-Graphics-Lab
C# WinForms lab project covering 2D graphics and animation. Includes geometric primitives drawing, star shape generation, and a real-time solar system simulation with planetary orbit trajectories.

This repository contains the source code for three laboratory assignments developed using C# Windows Forms. The projects demonstrate the use of GDI+ for rendering graphics and animations.

## How to use
Each file contains the necessary logic for the respective UI forms. To run these:
1. Ensure you are using Visual Studio.
2. Add these files to a Windows Forms project.
3. Note: Ensure that the UI components (buttons, textboxes, panels) referenced in the code match the names in your Form designer (e.g., `button1`, `panel1`, `textBox1`, etc.).

## Technical Details

* **Platform:** .NET Framework (Windows Forms)
